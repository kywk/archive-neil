+ NIKON GP1 FIELD TEST & WINTER SNOW IN TAIPEI +

入手GP1以後，趁著米米週末下午在台北市晶華飯店外面廣場有表演活動





順便測試一下GP1在戶外的定位能力





GP1在室內的定位能力是不太好的，我在家中開機好久，都沒辦法抓到衛星





一直到我忘記我有開機這件事以後，他才終於定位成功 = =|||





不過在戶外倒是蠻快的，我想大概還不到20秒吧，GP1就已經定位成功囉 ^^





只是說在高樓林立的台北，GP1的準確度如何呢?





這直接關係到月底我們要去香港，到底可以期待多少的問題 @@





以下就提供8張照片，來看看GP1的能耐嚕





另外就是那天下午拍著拍著，台北市居然下起雪來了 @@!!!





有圖有真相~~~















這張定位到晶華旁邊民宅去了 = =|||















這張更扯，定位到民生西路馬偕醫院的後面去了 = =…..















這張拉回來了，不過拉過頭跑到老爺酒店前面馬路上 = =….|||















鐺鐺~~~終於回到了晶華飯店前面廣場 QQ















繼續維持在晶華飯店前面廣場 ^^y















又偏到老爺酒店去了…..Orz，是因為我在拍老爺酒店嗎?















持續定位老爺酒店中…..(翻桌)















最後一張又回到晶華飯店前面廣場了…..T_T



另外我敢發誓這張我絕對沒後製 =  =+











OK….GP1定位成功率是…..





最精確 3/8 = 37.5%，100張大概最多40張準…..





次精確不要偏太遠 7/8 = 87.5%，也就是偏到馬偕醫院那張除外…..





真不知道是該喜還是該悲了 @@

+ LOLO & GP-1 OPEN BOX +





敗家 XDD








自從北海道之旅回來後，就一直有想要把自己拍的照片的位置紀錄下來的想法





若真的要做這件事，首選大概就是Nikon自家的GPS紀錄器 —- GP1





可是網路上討論GP1的文章不多，而且經調查發現





GP1是吃相機本身的電 @@…





冷開機到定位有可能要花上一分鐘以上的時間 ~"~





價格實在是不曉得在貴什麼 Orz





因為他是接在原本接快門線的孔，所以還要再去買另一組快門線 = =|||

所以除非記憶力極強，否則過個幾年，照片的確實地點在哪裡早就忘了





這大概就是我想買GP1最大的原因了

+ 武陵。風景 +





一些隨拍的風景稍晚上菜囉 ^^













開了好久，也睡了好久，我們終於要進入武陵囉 ^^













人家說行車時頭手不要伸出車外，我是都伸出去了











進入武陵沒多遠，我們就被風景吸引，準備下車走走，我一開車門就看見今天第一抹紅 ^^



















































































































其實我自己並不是很清楚，第一個點地名叫做什麼 @@



大概是因為在國內的關係吧，國內的旅遊我大概都不會太在意地點名



反正比較容易到達 ^^ 國外的話我就會很注意了，因為事前也會做很多的規劃



到達這個不知名的地點已經是中午，頂光卻還是一片藍，這就是海拔高加上空氣乾淨的關係囉



由於最近拍人像可能比較希望有光影或者顏色的對比，所以不知不覺連拍風景也被影響了



(當然除了shine兩口子那張例外，那張叫做搞笑側拍照，搞個枯樹當背景，是要後製成哈力波特嘛 XDDD )



































































接下來我們到了渡假村附近，先停了車，走路到溪邊櫻花勾吻鮭的保育地



光線已漸西斜，穿過樹林縫隙間的陽光，讓舉目所見更有層次，就連溪水也清澈見底



一伙人在這裡隨意的抓拍人像，風景，最後還拍起鳥來



我還是繼續我的光影與顏色的遊戲 ^^























































等到太陽終於被山擋住後，我們一行人回到渡假村 check in



武陵富野渡假村真的是怕遊客無聊，住宿安排DIY竹筒飯的行程



還在走廊間設置了一堆益智遊戲



米米運氣很好的終於拼成一個正方體，而我只是隨便亂拼成一個好像尼斯湖水怪的東西



晚餐後原本想出來看看銀河，卻因為月亮太亮了，導致星星反而黯淡…..





















































第二天天氣依舊晴朗無雲，離開渡假村時，溫度計還顯示24度哩



我們緩慢的出發前往福壽山農場，可惜福壽山農場的楓葉狀況沒有武陵的好，遊客也很多



我們在農場旁的小吃攤隨便解決了午餐，路邊的野貓一直對我們凹凹叫 @@























































































離開福壽山農場後，我們繼續往高處走，前往算是本次旅行最後一站，天池



天池的光線，也是這趟旅行我覺得最美的



錯落在樹梢間，完完全全的光影，既柔和溫暖，又有足夠的強度



這種光線，怎麼拍都美，不拍也很美囉 XDDD



離開這裡後，我們就慢慢的下山，結束這趟難得的旅行



臨走前築楓的最後一拍，正好陽光也灑了下來



此時我除了覺得山在歡送我們以外，更覺得



【怎麼這時會是築楓站在那裡……XDDDDD】











感謝築楓安排行程，兩位車長辛苦的駕駛



其實我們後來還有再往更高一點去，景色更純粹，只是光線已經不夠美了



這也是明年想要開三天兩夜，武陵一路殺到埔里團的緣故



期待明年11月底12月初囉 ^^y



感謝賞圖 <(.)>

+ 我們在武陵5℃ +





話說攝影師的宿命就是沒有自己的照片………….







自己的部落格也是，PO了好多篇，自己的照片好像沒有 = =|||



所以難得放上合照啦，1204-5由築楓帶團上武陵



還去了福壽山農場，梨山，據說差一點要開到清境了



這次有陽光男孩築楓在，一掃上次上清境的爛天氣



天空一整個爆藍呀~~



團員阿翔跟女朋友還拍起情侶照來，真是閃瞎我也 @@!!!



武陵富野渡假村真的是一個前不著村後不著店的地方



我們週五下午到渡假村附近，把附近的景點拍完後



太陽也差不多下山了，氣溫也開始急速下降(白天可是有24度呢!晚上就只有上面那張那樣….)



在渡假村用完晚餐後(坐圓桌吃把費，超怪的)



真的是沒事可做，所以跟米米一起去進行30分鐘的按摩囉



抒壓的感覺真好 ^^b



第二天早上我們就出發前往福壽山農場



真的是芭樂點人很多，楓葉的狀況也沒有很好



不過只要空氣好，對我來說哪裡都是天堂的啦 ~~~~



坐在路邊邊曬太陽，邊來一碗小吃攤切仔麵，真是一種奢侈的幸福 ^^



下午我們前往天池，約海拔2500公尺左右



彷彿越高，一切就越美



當一行人在天池旁的樹林裡拍米米時



我邊排隊等咖啡，邊看著陽光穿過樹梢，灑落在爬滿青苔的石頭上



遊客來來往往，每個人的髮絲光都是那麼的柔和又耀眼



我們再往高一點的地方去，一台藍色的野狼125，靜靜停在一片高山草原的旁邊



這絕美的暮色，深深印在我的腦海裡



(所以我要去買GP-1啦，這麼美的地方怎麼能不在我們的旅行地圖上紀錄下來 =皿= )



我們乘著翻過山頭的雲霧，不情願的回到都市裡……..









明年揪一團武陵到清境的好了，三天兩夜一路開過去 = =+

+ 清境。人像。風景 +





清境，有羊 ^^









自從去年退伍後，上來清境住了幾天，便下定決心每年都要上來一次





乾淨而且冰涼的空氣，一直是我的最愛





這裡的民宿大多充滿歐式風格，更讓整個清境像童話世界一樣





如果有天真的海平面上升，我們現在住的平地都被淹沒的話





我想我會馬上逃到清境吧





去年上來時，都是超美的藍天，今年卻見到了完全不一樣的清境





飄小雨，不知道是山嵐還是霧，一直盤据在山頭，日夜不散





又是另外一種感覺，期待明年的清境會給我什麼驚喜囉 ^^



































































今年上清境前，順道繞到奧萬大去，那時天氣還很好



不過時機點太早了，所以當然沒有楓葉囉…….











































離開奧萬大，到達清境已經快下午兩點，吃飯前還有藍天



沒想到吃完飯後，整個清境已經被霧籠罩



不過遊客觀賞剪羊毛秀的興致還是不減 XDDD



每次陽光露臉的時間大概只有幾秒鐘，所以也就沒有拍什麼了………





















































這次住的地點是，香格里拉空中花園



去年發現他有調酒吧，但去年因為正在整修，所以沒喝到



想說今年一定要喝到，所以上來前就二話不說定了這家



結果！！！今年調酒師因為去旅展幫忙，不在山上……Orz



























整日的霧，讓我們想看銀河也沒機會了



只能記錄下這個灰濛濛的清境夜晚，外面的溫度大概是12度



民宿裡住滿了人，來來往往，好像一個家庭



晚上跟米米一起窩在房間看HBO，這就是我要的悠閒吧 ^^
