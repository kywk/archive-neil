【活動報導】在一個夜黑風高的夜晚.....





在一個夜黑風高的夜晚….雲層漸厚….



失去月光的照耀….大地進入了一片黑暗….連樹木都顫抖著….感到不安…..



彷彿有什麼事情即將發生….有什麼不知名的物事即將出現…..







果然…當月亮緩緩失去光芒時….可怕的怪獸出現了….



同一時間….怪獸的背景音樂也響起…..

究竟誰能消滅怪獸，拯救這個世界呢？欲知詳情，請見下回分解~






上面是來亂的 XDDDDDDDDDD

可惜昨天的天氣似乎不像氣象局預報的，受高壓籠罩萬里無雲



反而有很多雲層來來去去，有一大半時間太陽是被雲遮住的

不曉得埃及有沒有像神鬼傳奇第一集一樣，大批蝗蟲鋪天蓋地而來 XDDDDDD

【旅行。趣】初夏的北海道 -- 後記

從北海道回來已經快一個半月，整個旅遊的紀錄也要告一段落



這篇打算放一些從北海道帶回來的一些文件、地圖，以及四天三夜之旅最佳護身符 — JR PASS







上面就是我們的 JR PASS，以及另外指定劃位的指定券



北海道JR公司是允許購買PASS的旅客可以無料劃指定席的，但前提是該列車有指定席可以劃



在出發前，我們已經將可能搭乘的車次都列在行程表中，並且也預計好最有可能搭哪一班



同時，我們也以為所有的車次都可以劃指定席 @@



直到我們到了新千歲空港JR MIDORI櫃臺，讓可愛的紅色制服櫃臺小姐幫我們處理換PASS的事情時



才知道，原來我們預計搭乘的車次，到小樽的來回車都是自由席，旭川到美瑛的車次也是自由席



只有新千歲空港來回札幌的機場線，及札幌來回旭川的車次有指定席可以劃



所以指定席的票券就只有札幌來回旭川的了



JR PASS的使用也很有趣，持PASS是不能走磁卡機的



我們第一次使用時以為可以用磁卡機，結果吸進去後被吐出來



後來才知道要走最靠近服務台的小門，並且在經過時亮一下PASS給站務人員看就好



這種場景是不是有點熟悉呢？是的！我們那四天就像下圖這樣，很搖擺的給他走過去XDDDDDDDD







那麼指定券買來有什麼用呢？答案是車上查票用，JR查票人員真的就像出發前google的結果一樣，查的很勤 XDDD



JR列車上似乎是可以飲食的，我們在旭川前往美瑛的路上，就吃了早餐



不過札幌市營地下鐵好像就沒看到有人在吃東西，所以我們也不太敢冒著被白眼的危險…….



JR = 台鐵，札幌市營地下鐵 = 捷運，我們是這樣分類的吧 XDDDDDD







嗯？怎麼又有個JR PASS呢？這個是在台灣先買的EXCHANGE ORDER，到了北海道再把他換成PASS



由於是不是能在北海道當地直接買PASS，網路上實在是眾說紛紜，所以我們選擇比較安全的作法



新千歲空港站的章表示我們是在機場的JR櫃臺換的票 : )



換票時同時可以指定PASS何時開始啟用，因為我們買的是三日券，所以預定是5/29才開始使用囉



這就是為什麼PASS上面寫5/29-31



EXCHANGE ORDER找國內的旅行社或者Y拍找找，都有代辦







嘿嘿，這是第二天比較值得紀念的地圖 — 美瑛拼布之路PATCHWORK ROAD的手繪地圖，繪製者是松浦商店的老闆



話說那天下午我們決定去騎拼布之路，所以按照google大神指示，我們到松浦商店租腳踏車



到那邊的時間大概是下午三點多了，老闆問我們要騎拼布之路還是景觀之路



(松浦商店的老闆不會講英文，所以我們聽他的日文聽得有點辛苦)



確定我們要走拼布之路後，老闆從抽屜抽出一張地圖，並且開始講解哪邊有什麼東西



看到地圖上方有些被畫叉叉的地方嗎？那是老闆說「不用想了，你們的時間不夠騎到那邊的」 XDDDDDDD



老闆還很好笑的告訴我們，去不了的地方，到北西展望之丘 (中間那個被圈起來的三角形) 用看的



還示範了眺望的姿勢 (手舉起來手掌打平大拇指靠在額頭上) ，表示「別想過去了，用看的就好」



我們看起來這麼弱嗎 T__T



老闆還特別告訴我們哪邊有恐怖的上坡，不要靠近 Q_Q



真的很可愛，那天我們還車稍微晚了一點，回到店門口，原本停在店門口的腳踏車都已經收進去了



感謝老闆 <(.)>







哈哈，這是小樽浪漫散步地圖，忘記是在哪邊拿的了



這張地圖的東西南北有點混亂，印象中上方應該不是北方



由於小樽有什麼好逛的，事先我們google都查好了，所以這張地圖對我們其實沒什麼幫助



但因為實在是太可愛啦～～～～～～～～而且還是彩色的！所以當然要留下來囉



看著上面寫滿滿的景點，下次再到小樽，應該還有好多可以玩，也許去看看小樽的港口吧



或者沒有下雨的晚上去逛逛北方的華爾街，或者在雪季時前往情書的另一個點 — 天狗山滑雪場囉







最後是舊日本郵船小樽支店的入館券、北海道神宮的章，及H1N1給我們的紀念品



當時正是H1N1搞的大家人心惶惶的時候，可是機會難得，我們還是決意前往



在往北海道的飛機上，所有人就被要求填一些有關健康聲明與保證的文件



後來都被收回去了，只剩下黃色那張變成一個紀念



此行我們跑得點幾乎都不用門票，所以僅存舊日本郵船小樽支店的入館券留著



北海道四天三夜之旅的紀錄，終於在本篇後告一段落了



據說明年還有紐約十天行與義大利十天行 @@|||



姑且不論假是不是真能請那麼久，四天三夜遊記就寫了一個半月，十天是要寫半年嗎 = =|||



希望明年的幾個旅行也能順利前往囉



旅行，真是好物呀…….: )

【旅行。趣】初夏的北海道 DAY-4





已經回來一個月多了，終於到了最後一天……= =b










來到北海道第四天的一早，我們按預定的計畫，一大早就準備退房，不會再回來這裡了



因為要去新千歲機場，要在JR札幌站坐快速Airport，所以我們預計把行李寄在車站寄物櫃



清早起來，收拾行李之後，為我們三天晚上住的小小房間留下影像紀念



小，但是清爽乾淨，第一天晚上為了ADSL網路，飯店人員還在房間陪我們弄了好久 : )



出去玩要住豪華旅館嗎？其實不用，小小的旅館反而有種流浪的感覺











一張是門口，一張是飯店的Lobby



第一天傍晚checkin的時候，一進飯店大門就聞到花香，後來，這股花香就陪伴了我們四天



直到這天我們要離開了，門口還是一樣的香氣，一樣的沒看到花在哪裡



Lobby前擺設了一些玻璃工藝品，還記得第一天到時，我們先到右邊的沙發放東西



然後商量著要怎麼跟飯店人員溝通，後來每早出門前去寄放鑰匙，跟每晚回來去拿鑰匙



也能夠很自然的用簡單的話打招呼，他們總是笑咪咪的 : )

















寄放完行李後，我們前往此行最後一站，北海道神宮



神宮位在札幌市營地下鐵東西線的円山公園站，出站時，天空已經開始飄起雨來



這樣也算不錯，短短四天，北海道的晴陰雨我們都體會過了



從円山公園站到北海道神宮的路上，我們經過了這間不曉得是餐廳、住家、辦公、還是旅館的建築



只覺得住在這裡面的人真是幸福 Q_Q







看到北海道神宮的鳥居，當然一定要拜一拜 XDDDDD



雨中的円山公園十分幽靜，幾乎沒有人，這樣也好，前往神宮的路上給人的感覺本該如此























円山公園裡面其實不只是北海道神宮，還有許許多多小小的神社，分別因為不同的故事而建立



小神社門口右邊是求籤處，沒有人看管，求籤者自己投100羊，求一隻籤



穿著藍色制服的小朋友，大概是戶外教學吧，在一片綠中格外明顯



透過屋簷縫隙撒下的柔和光線，讓這兩塊布的圖案更顯莊嚴肅穆，不自覺的想要拍下來











走著走著，我們到了神宮的門口，恩？怎麼門口停了幾台紅色的車子，而且門口往裡面一點點站了一群人？



走近一看，原來今天正好有新人來神宮舉行婚禮，所有人穿著傳統服飾，在神宮門口拍團體照，真的好溫馨 : )



















趁著新人拍照空檔，我們從旁邊的小門溜進去，雨天的神宮多了一份寧靜的感覺



一樣著傳統服飾的神宮工作人員，匆忙的來來去去，大概是在為待會兒大殿的結婚儀式做準備



不管是紅色或是藍色，在這片風景裡都是那麼的協調，連雨傘彷彿都充滿了堅持呢











我們沿著神宮四周的迴廊，從神宮門口走到大殿前，此時原本神宮門口的團體照拍攝已經結束



遊客也開始陸陸續續進來，我們的目標則是欣賞日本傳統的結婚儀式











好少人的結婚儀式，比起台灣，真的是大大大的不相同



但也許就是這樣的簡單卻隆重，比起台灣人老是愛辦桌，還要來的更謹慎，也更得來不易而該珍惜



原本只是看，沒有打算要拍的，總覺得自己不是他們請的婚禮攝影，他們又感覺這麼的莊重



即使可以偷拍，他們也不曉得，但總覺得不應該如此



可是看著看著，旁邊居然有日本遊客拿出小相機拍了起來 @@|||



所以我也偷偷壓了一張，我沒原則阿阿阿阿阿阿阿阿阿阿阿～～～～～～ Orz























離開北海道神宮，也快要接近中午，我們必須在兩小時前至機場checkin，所以一路趕往札幌站



往札幌站的路上，我們搜尋著札幌著名的湯咖哩，希望運氣很好在路上正好可以碰到



可是天不從人願，我們此行跟湯咖哩似乎沒有緣份了，為了不耽誤飛機，我們還是吃了拉麵，完成每日一拉麵的壯舉



味之時計台是網路上來到札幌必吃拉麵，不過跟青葉比起來就真的還好了，有趣的是他們吃玉米還有一個特別的湯匙



一樣的料多湯濃郁，一樣的大把翠綠蔥花，這就是北海道拉麵，一個我回到台灣還一直懷念的好味道















此時，我們已經在登機門口，即將踏上歸途，暌違十年的國外旅行，也即將告一段落



四天實在過於短暫，讓我們覺得以後一定還要再來



四天的回憶卻很豐富，填滿了我們的記憶





還有後記喔～～～～ : )

【旅行。趣】初夏的北海道 DAY-3





先來張小樽必拍芭樂照 XDDDDDDD






來到北海道的第三天，我們的行程是小樽，岩井俊二【情書】拍攝的地點



因為昨天的美瑛行實在累壞了，所以我們放棄了原本搭一早的JR到小樽的計畫



決定先在飯店吃個早餐，不要再那麼趕



所以我們到了小樽，差不多已經是上午九點多囉







上午的行程是手宮公園，情書中男藤井樹騎著腳踏車經過一個下坡



將牛皮紙袋套在同樣騎著腳踏車的女藤井樹的頭上



傳說中的那個下坡，就在手宮公園，我們預計搭小樽巴士前往



JR票當然是不能用於小樽巴士的，所以我們每人付了240羊，一路前往手宮公園











其實我們並沒有找到手宮公園的門口，手宮公園位於山丘上，我們是沿著山腳，隨便找一條路上去



沿路經過半山腰的住家，還經過小小的神社，太陽依舊猛烈，我們邊走邊找往手宮公園的路















經過一連串上坡，我們真的進入手宮公園的範圍，可是，情書拍攝的季節是冬季



我們是初夏前往，再加上對手宮公園又不是那麼熟，要找到那個傳說中的下坡，當然是難上加難



所以，我們挑選一條下坡旁的一片草地，把這裡當作是男藤井樹當時的起點，向下眺望



「お元气ですか〜？」「私は元气です〜！」  電影裡博子是在天狗山的滑雪場喊著這兩句話，向樹道別



我們因為行程的關係，所以在這片草地上喊著











離開手宮公園的途中，意外發現這一長片落葉堆，落葉堆非常深且柔軟



人一踩下去，會一直陷到腳踝都不見了呢 :)







離開手宮公園後，我們步行前往下一個情書拍攝景點 — 舊日本郵船小樽支店



路途中看到這台可愛的巴士，潔思米立刻就吵著要合拍拍～～～



仔細一看車長還坐在裡面呢！還好他沒突然把車開走 Orz







跟可愛的巴士合照後，我們進入舊日本郵船小樽支店，這裡是傳說中長大的女藤井樹工作的圖書館的景



一進去便是古色古香，天花板垂吊下來的一盞盞燈，有點像星星



這樣感覺的地方，應該是我們此行第一次遇到，這也顯示我們更接近有很多古老建築的小樽



















整個支店並不大，只有一二兩層樓，可是走進館內沈靜的感覺，又跟在外面陽光普照的感覺大不相同



館內的服務員老伯伯熱心的跑來幫我們介紹，不過他好像只會說日文的樣子



我們是整個有聽沒有懂，後來，老伯伯拿給我們繁體中文的介紹



我們才知道，館內一定要看得是這片金碧輝煌的壁紙，由於年代久遠，四周的壁紙很多都已脫落



只有這片還算完整，搭配著古老的宮廷椅，有種繁華卻又已退盡繁華的感覺







但是，就這樣晃了一圈，情書中圖書館的場景，究竟在哪裡呢？



明顯的館內根本沒有書架這種東西，正當我們納悶，並且回到門口準備離開時



才發現，門口售票的櫃台好眼熟，原來，這裡就是長大的女藤井樹在回信給博子



女藤井樹的同事問，妳到底要不要來幫忙啦？的那個窗口



想必當時中山美穗就是坐在這個小房間裡，回著信吧











離開舊日本郵船小樽支店後，我們再搭乘巴士回到小樽市中心



肚子已經餓得咕嚕咕嚕叫，當然是前往小樽有名的壽司通，吃我們除了拉麵以外的午餐



本來想要去小樽有名的政壽司，也已經走到門口，不過實在因為政壽司太貴而作罷



後來我們索性往運河方向走，依照旅行書上面的介紹，找到這家握なかSUSHI



(其實也沒比政壽司便宜，這邊的壽司店價位都差不多……Orz)



那…..怎麼沒有壽司的照片呢？因為當時被價錢震撼到，加上又很餓，所以壽司一來就忘記拍，直接吃光了XDD



整體來說其實他們的握壽司還好，台北也吃的到這種程度的，算是有點失望 = =



















小樽堺町這條路上，聚集了網路上來小樽必推的景點與名產店 XDDDDDDD



北一硝子、音樂盒堂、浪漫館、銀之鐘、北菓樓、LeTAO、雪印，還有好多古老的建築都在這條路上



我們沿著這條路走，有興趣的店就進去逛逛，路上常常可以看見馬車與人拉車



一條小小兩線道的馬路，聚集了所有來到小樽的觀光客 : )



我們邊走，邊吃超級好吃的冰淇淋 (好像吃了三支XDDDDDD)











其實堺町沿路的商家，大部分都因為做觀光客的生意，進去便是玻璃或手工藝品



價錢都不便宜，店家內其實也沒什麼好拍的，唯一我們有興趣的便是北一硝子三號館



在網路上看別人拍的照片，館內是那麼美，卻也查到似乎不能在館內照相(只能偷拍，拿D3怎麼偷拍 = =||)



還好，當我們真的到了三號館，才發現原來是禁止用閃光燈而已，不是不能拍



此時就是高ISO超強的D3的天下啦～～～～～～～～～～～～～



另外，其實三號館是一個咖啡廳，也賣些糕點、冰品，他們的紅豆湯圓牛奶冰超好吃的啦 ^^b











離開北一硝子後，我們繼續往Meruhan十字路前進，途中經過我這次小樽行必至的北菓樓



為什麼說必至呢？其實小樽有什麼好吃的好看的，大概行前拜請估狗大神都查的差不多了



堺町沿路上當然有很多好吃的糕餅點心，可是唯一在還沒出發前，就讓我在台灣對著電腦流口水的



就是北菓樓的妖精之森年輪蛋糕！！所以看到北菓樓就馬上衝進去了 XDDDDDDDDDDDDD



基於背的東西實在已經很多，而且新千歲機場裡面什麼都有得買，所以我只是打算進去試吃一下



然後呢…………..我就一直試吃…………………….>/////////////////////< (所謂一直大概就是我吃了快十次……)



還好日本店家真的都很有禮貌，也沒人白眼我，就讓我當一次沒水準的觀光客吧 Orz















LeTAO與銀之鐘咖啡就沒那麼重要，所以LeTAO我們只是經過，連門都沒進去



不過他們的綠色大門真的是蠻有特色的，很搶眼，可惜遊客太多，一直抓不到背景沒遊客的畫面



銀之鐘的賣點是喝完咖啡可以把咖啡杯洗一洗帶回家，不過咖啡實在不怎麼樣，昨天在北工房喝的咖啡好喝多了 = =



在銀之鐘喝咖啡的同時，外面的天空也開始飄起雨來，而且有越下越大的趨勢



我們決定喝完後啟程回到小樽運河著名的倉庫區附近，等待日落後運河旁的燈點起來











回到小樽運河倉庫區的途中，我們經過色內十字路



色內十字路也是情書的點之一，在這個有小樽郵便局的十字路口，長大後的女藤井樹與博子在此碰面



其實也不算是碰面，博子即將離開小樽，在這個十字路口正好與騎著腳踏車的女藤井樹擦身而過



兩個容貌一模一樣的人，在這個十字路口有了短暫的重疊



我們到了這個十字路口，開始研究樹是從哪個方向騎腳踏車過來，博子當時是站在哪裡



包括考慮到樹應該是從山上的家下來，電影中路燈的樣子等等因素，但結果還是 ————————- 研究不出來…………Orz



所以我們在這裡留下了幾張影像，隨著雨越來越大，我們趕忙前往運河倉庫區



並且在運河倉庫區的一家德式酒吧休息 (兩個人又都睡死…)，等待運河燈亮起，北海道第三天的行程，在大雨中劃下句點…..











回到札幌後，我們在札幌站地下街找了一家拉麵，當作晚餐



原本是要找札幌著名的湯咖哩，可是已經很晚了，我們也不曉得湯咖哩在哪裡



只好看到拉麵又走進去，免得拖到更晚沒東西吃



這次吃個是地下街的「華龍」拉麵，可能是昨天被青葉拉麵震撼到，這家感覺就還好…..



有趣的是在吃的同時，陸續有好幾個穿著西裝，看起來就像是在某某商社工作的年輕人



可能是下班了，回家前也進來吃碗拉麵



那種感覺好像走進日劇的場景一樣，真實的呈現在眼前



也許我們真的受日本文化影響太深，到了日本，總是有種走進電視畫面裡的感覺吧 : )







小樽街道的異國風情，巧妙的與日本人給人的感覺相融合，傍晚的雨更讓運河透出一種朦朧的美，這是一個充滿思念的城市 : )

【旅行。趣】七美。全輯發表





6/24更新






從開始報名到出發，中間經過三個多月(吧？)的七美望安澎湖行，終於在上週末出發



北海道回來後，接著期盼的就是這次活動，隨著出發時間的接近，柯柯也終於在Blog宣布「大家七美見哩…..」



19號一大早，所有人聚集在松山機場，準備分成兩班飛機前往馬公，再一起搭船前往七美



雖然一路上天氣都陰陰的，甚至在馬公的遊覽車上與往七美的船上甚至下雨



不過到了七美後，下午的狂爆藍天與真的會咬人的陽光，讓所有人怎麼想也想不到我們將受困七美四天XDDD



其實，這是我第一次參加這麼大團的外島團，以前總耳聞外島是多麼美，每年出團必有求婚大作戰



沒想到，七美固然真的很美，純樸充滿台灣味的居民，一望無際的海天一色，隨隨便便就出景的夕陽



讓人目不暇給，讓我們在第一天還風平浪靜時，不顧炎熱騎著機車試圖遊遍島上的每一個角落



除此之外，居然本團也有求婚大作戰，這好像真的是外島團的傳統，每年都會有情侶求婚 @@



在七美的第二晚，所有人暫時忘記了因颱風無法繼續行程的遺憾，真心話大冒險玩到民宿屋頂快掀起來



並且所有人偷偷的配合原訂計畫，讓元宏的溫馨又很拼體力的求婚計畫順利完成 XDDDDDD



持續滯留的第三天，雖然風雨交加，但當民宿老闆娘送來熱騰騰的海鮮粥時，所有人都齊聲歡呼(真的快要變難民)



最後，終於在Markur與柯柯的大力奔走下，我們可以趕在週一晚上回到台北



第一次的外島團，卻好像什麼都有了，有美麗的風景，極盡搞笑的大師群，有求婚，有颱風



四天沒有電腦、網路、msn、email的日子，卻是難得的經驗



感謝柯柯、Markur辛苦的主辦，偶而放逐，遠離原本的生活，也是一種生活的方式囉



北海道還有兩天沒po，看來圖要欠很大了 Orz，先來個文字記錄，免得到時想寫也寫不出來



期待眾位大師的作品，小弟慢慢龜圖去 XDDDDDDDD
