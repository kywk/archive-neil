[ 一家四口 ] Kaloling x Arjer x Babies





拍了這麼久，最近才慢慢覺得，攝影是不是應該回到幫家人，朋友服務



留下真正是自己最親密的人的影像，而不是一味的拍美女，拍妹，拍所謂的人像作品



當然，以攝影為職業也是相當可喜可賀的事，最近也有好幾個認識的朋友相繼拓展攝影事業



但我只是突然覺得，不能讓硬碟裡只有跟自己不相干的人的照片一堆






這次有機會跟潔思米的姐姐姐夫，還有兩個小朋友見了面



在拍攝的過程中，我看到一個父親只是專注的用小相機在捕捉自己的寶貝



一個母親滿足的看著父子倆



我敢說他們的心裡是滿足的，絕對不會像我以前拍那麼多妹以後，反而空虛



或許



如果不是拿攝影當職業，真的應該多拍拍朋友家人



如果拿攝影當職業，更要撥出時間，拍拍朋友跟家人吧 : )



















































































































我也要去多拍我的家人，決定過年帶爹媽去度假吧 = =+

【電影心得】聽說 Hear_Me -- 好雷不斷^^





最近在批踢踢Movie板好雷不斷的愛情小品 — 聽說 Hear_me








尼尼一直有去批踢踢movie板閒晃的習慣





蒐集別人看電影的心得，再決定到底要不要進去電影院看





而 聽說 Hear_me 在前一陣子攻佔movie板板標





所有去看過的鄉民們，也大多都是好雷





所以某次晚上正好有機會，我們就趁著他下檔的最後一週，趕快進電影院支持好電影 ^^









其實男主角彭于晏主演的電影，好像已經看過不少部了





最近大概就是 愛的發聲練習 跟 聽說　兩部





愛的發聲練習有太多值得思考的東西，導致想寫網誌





可是總覺得只看一次，寫不出來，只好一直擱著 @@





聽說的劇情就簡單多了





天闊 (彭于晏飾) 是一個燒臘店的小孩，在某次送便當的過程中認識秧秧 (陳意涵飾，上面海報的女生)





秧秧的姐姐小朋 (陳妍希飾) 是一名聽障游泳選手，立志參加聽障奧運





秧秧為了讓姐姐小朋能專心在奧運訓練上，獨自扛起家計，四處打工維持姊妹倆的生活





這樣堅強，認真，溫柔的秧秧，吸引了天闊，因此展開天闊很宅的追求 XDDDDD





本片的重點在於，因為天闊與秧秧第一次見面時的場景時間，所造成的一個美麗的誤會 —





天闊與秧秧都誤以為對方是聽障人士，但其實兩個人都不是





劇情很巧妙的讓觀眾一開始就知道天闊並非聽障人士





但在最後才揭開，其實秧秧也非聽障人士





兩個人在這中間透過手語的交流，反而凝聚的情感，比用嘴巴說的，更加深刻而堅定





也讓所以觀眾感受到，無論是生氣時，手的動作越來越快，越來越激動





或在感動時，那樣的平緩，溫柔的手勢





都激發出更豐沛的感情，一波一波的，衝擊所有觀眾的心………..









劇情就不在多透露了，讓我最印象深刻的





第一個是秧秧在東區街頭藝人表演時





相信大家去東區逛街看電影等等，都會看到像雕像站立不動的街頭藝人





你必須投錢，他才會開始動作





而聽說，巧妙的將這樣的職業融合在電影的腳色裡





我相信，所有看過聽說的人，再回到東區街頭





將對這些街頭藝人，多了一份感情





第二個場景當然是天闊爸媽好像相親似的，與秧秧見面





那份愛護天闊的心，以及願意接納秧秧的包容





再加上很耍寶的紙上對白





真的是讓人笑中帶淚，又哭又笑，哭笑不得呀……..





以下轉貼一些劇照，其中包括我最喜歡的街頭藝人的場景





可惜天闊爸媽的劇照不多………





聽說 Hear_Me ，一樣推薦給大家囉 ^^

【電影心得】The Time Traveler&#39;s Wife 時空旅人之妻





The Time Traveler's Wife， 時( ㄌㄨㄛˊ) 空( ㄌㄧˋ) 旅( ㄧㄤˇ ) 人( ㄔㄥˊ) 計 ( ㄐㄧˋ) 畫 ( ㄏㄨㄚˋ)











上面真的不是開玩笑的，那是我看完本片後，除了覺得好看以外，第一件想到的事 XDDDDD





其實我對時空旅行的題材一直很有興趣，從第四台一直在重播時光機器 The Time Machine，我就一直不停重複看





The Time Machine 敘述男主角因為未婚妻不幸車禍過世，為了改變這個事實，男主角發明了時光機器並回到過去





但不論他怎麼努力，未婚妻總是會以不同的死法，呈現在他面前





到最後，一句點醒男主角的經典對白　 "Each of us is a time machine, the past is memory and the future is a dream."





才讓他終於恍然大悟…………….





接下來的，班傑明的奇幻旅程 The Curious Case of Benjamin Botton，也是一部非常經典的時空系列電影





所以在蘿莉養成……對不起，時空旅人之妻上映時，我也在想應該去看一下





簡單的劇情如下









Henry (艾瑞克巴納飾)，從小患有時空錯置失調症





他會不由自主的從原本的時空消失，到過去或未來的時空旅行





旅行的地點，旅行持續多久，他都無法控制，他只知道他一定會回到原來的時空





這樣的他，在某次回到過去的旅行中，遇見了他將來的妻子，Claire (瑞秋麥登絲飾)





並且開始養成計畫 (其實真的是無誤)





後來，他們在原本的時空終於相遇，結婚，生下一個女兒，然後 Henry 在女兒五歲時





因為在時空旅行中受到槍擊，離開了人世間，結束了時空旅行的一生









在電影中有兩個部分讓我印象深刻





第一個是 Henry 與 Claire 結婚那天，原本該是 Henry 走上紅地毯，從 Claire 的父親手中牽過 Claire 的時候





Henry 卻又不由自主的時空旅行去了 @@





眼看新郎不受控制的落跑，婚禮將無法進行下去時





Henry 突然出現! 　只不過，這個 Henry 好像比較老 @@|||





原來，這是未來時空的 Henry ，剛剛好的旅行到現在正在進行婚禮的時空





也剛剛好的救了應該在原本的時空進行婚禮的年輕 Henry，完成了這場婚禮 XDDDD





一切都是那麼的注定，在牧師台前的 老Henry 與 Claire，不禁相視露出微笑





第二個是 Claire 一直很希望有一個 Henry 的小孩，可是因為 Henry 的時空錯置失調症似乎會遺傳給胎兒





因此導致胎兒多次還在 Claire 身體內，就時空旅行去了，造成每次懷胎都流產 T_T





Claire 因為這種情況，深受精神上與身體上的煎敖





在不希望妻子再受這樣的煎熬的前提下，Henry 瞞著妻子去做了結紮手術





手術完後才告訴 Claire ，Claire 傷心不已





然而，Claire 後來還是懷了 Henry 的女兒，原因是某次





過去時空的 Henry (尚未結紮的)，旅行到現在的時空，現在時空的 Claire 又再次懷胎 @@





我還記得現在時空的 Henry 知道這件事時，那個傻眼又拿老婆莫可奈何的表情 XDDDDDDD





所幸這次的懷胎，反而順順利利的生下了小女兒，一個同樣遺傳父親時空旅行的能力，卻比父親更能控制的女兒 ^O^





雖然時空錯置帶給這兩人很多痛苦與忍耐，可是也同樣帶給他們更多的幸福與珍惜





( 當然也帶給他們很多錢……這大家用想像的，或許也猜的到為什麼……Orz )





這是一部很溫和又淚中帶笑的的電影，現在應該早就下檔了





不過還是建議大家可以看看囉 ^^
