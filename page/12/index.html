我們在魔幻之都尋找似曾相識的風景 -- 2010元旦港澳之旅 Day1 下午





在翠華結束到香港的第一餐後，我們出發前往香港島的南方 — 赤柱





























在規劃香港行程之初，我們考慮在赤柱與淺水灣間選一個



後來選擇了赤柱，是因為淺水灣似乎比較多人去過，而且可能只有海灘



而赤柱除了也是海邊外，還有赤柱市場，香港的市集也是我們此行的重點



搭雙層巴士到赤柱其實蠻快的，車程大概也才花三四十分鐘



從上環一路經過香港的商業中心金鍾，在一路往南邊經過香港的高級別墅區



景色已從夾道林立的摩天大廈，人潮洶湧的大馬路



逐漸轉向可能只有兩三層的別墅，窄窄的僅容單線雙向的山路，以及路旁成排的樹蔭



變化很大，卻還是香港













好像是啥中國人民解放軍大廈























迅速通過高等法院，還好DDD夠快 XDDD













第三天我們會搭這個去時代廣場 ^^













香港其實真的很多…..看起來像外勞的外國朋友，不過我真的很懷疑他們是不是外勞，還是是觀光客



注意看左下角那位還拿LV包哩，在台灣我可從來沒看過外勞朋友這麼高調













右邊那棟是香港賽馬會，其實本來此行有要考慮進賽馬場看看，畢竟那也是特色之一



不過後來沒去，沒時間 T__T







































穿過隧道，我們已經慢慢離開都會區，沿著香港島南邊的海岸線，一路抵達赤柱







































雙層巴士沿著海岸線行駛，我原本以為台灣的公車司機算技術非常好了



沒想到香港的雙層巴士司機真是讓我五體投地的佩服 @@



一路往赤柱的路上我們經過香港的高級別墅區，別墅區的路都很窄



而且沿山坡建築，一路高低起伏，彎彎曲曲，可是我們的雙層巴士司機居然像飆車一樣



我跟米米運氣很好，坐到上層的車頭位置，好幾次眼看車頭好像快撞到人行道的護欄，卻總是差幾公分



至於兩旁長得太茂盛的小樹枝就比較倒楣，司機一點也不在乎的就衝過去 XDDD



甚至司機還在一個狹窄的T字路口瞬間靠左，然後一個360度大迴轉再往原路回去



也是一樣，眼看迴轉時快要撞到路邊的標示桿，卻又硬是擦過去



而且重點是司機好像老神在在，知道絕對不會撞到，幾乎沒特別再踩煞車 @@



就這樣經過特技表演，我們在赤柱廣場下車



在一路步行經過兩旁都是餐館的赤柱大街 (東西非常貴，我們都吃不起 Q___Q)



往赤柱市集前進



























































走過赤柱大街，我們到了市集的入口處，米米馬上在這邊解決紀念品的問題，我則是東看西看，順便偷拍



應該是因為假日的關係吧，整個大街與市集人都非常多，而且似乎各國人都有 @@



接下來我們進到市場更內部，整條街更是人擠人，有點像假日的九份老街



不過九份早就太商業化了，無聊的很，赤柱市場還是有很多很有特色的店 ^^



























































赤柱市場大概最多的，除了傳統服飾或布料的店外，便是畫攤了



各式各樣的畫，香港的夜景，街景，都在這裡





























有趣的扛棒又來了 XDDD



在香港另一個我很喜歡看的，就是他們的電影海報，特別是翻譯



裘德洛 => 祖狄羅，鋼鐵人 => 鐵甲奇俠，即使翻譯也仍有濃濃的港味 XDDD



接下來這個西醫更囂張，內外全科醫學士，不仔細想想還以為他都能看哩，不過我想應該是指我們的家醫科吧



家醫科的確是什麼都能看，可是要更專業點還是要個別科呀 XDDD



就這樣，我們在赤柱的巷弄裡穿梭尋找新鮮事，度過到香港第一天的下午



然後一樣坐上雙層巴士，回到上環，晚上我們即將一覽百萬夜景 ^^

我們在魔幻之都尋找似曾相識的風景 -- 2010元旦港澳之旅 Day1 上午





終於在漫長又發霉的年假中，把香港的圖修出來 Orz



(年底真的很忙嘛)











北海道一行回來後，米米和我就在計畫下一個地方要去哪裡



畢竟自助旅行是會上癮的 XDDD



好不容易元旦有三天長假，我們便決定到香港澳門過新年



港澳幾乎就像台北飛恆春一樣，沒花多少時間，飛機就已經落地



所以連睡覺都省了，下了飛機也也沒有什麼特殊的感覺



一直到靠近海關，我才因為港澳對於事物的特殊名稱，終於感覺到我踏進了香港



一個我闊別十年不只，印象中上次來是住在一個小小公寓，有著窄窄的走廊的地方























海關大堂



其實我覺得很多我們倒背如流的港片，例如星爺系列啦，無間道系列啦，還有其他很多系列



第四台都應該播放香港話的版本，因為他們的語言，有時總是更直接且貼切的描述一樣事物



海關大堂，確實挺大，而且大堂好像有點包青天要"升堂~"的感覺 XDDD







































接機大堂 XDDD



聖誕節的裝飾還在，據說香港是一個非常有過節氣氛的地方，包括聖誕節，元旦，很多人都選擇來香港跨年



我沒有這個幸運(也沒這個錢跟假)，只選擇在元旦早上踏進這個地方



不過有朋友更慘，來跨年卻得了重感冒 XDDD







































活力 ^^



我想是要來到香港的人，都能夠很有活力的吃吃喝喝血拼吧 XDDD



我們要搭乘活力快線，一路往香港島去



在香港除了吃吃喝喝逛逛以外，會讓我們很high的就是"熱點"了



所謂的熱點，就是Free的Wi-Fi囉~ 看~米米的唉俸在香港也可以無線上網呢~~



不過要小心挑Free的熱點使用，以免回台灣收到天價的漫遊帳單，那就欲哭無淚囉

















































領到機場快線的票，我們一路往快線的方向



可能是該來跨年的人潮前幾天就到了吧，整個機場空蕩蕩的，沒什麼人



這就是我們原先預想的，我們不要人擠人的痛苦行程

















































在快線車廂內，一路上的風景



快線的行進標示，是用一顆一顆的藍色小圓點組成，區別站與站間需要花費的時間，圓點的數量也有不同



一路上經過海景與各個中繼站，快線各站很貼心的在月台放置行李推車



還幫你排的好好好，香港不愧是群聚各國人民與觀光客的地方



不曉得是天氣陰還是車窗是藍色的，拍出去是一片藍

















































闊閘機 ^^



繼續拍一些奇奇怪怪，可是卻形容的非常貼切的標語



常坐台北捷運的人一定也知道台北捷運也有類似較寬的閘門，可是似乎沒有特別標示



我們原本預計搭接駁小巴到酒店，但卻找不到蘭桂坊的接駁車(之後才知道是跟另一家酒店同一條線)



只好去坐地鐵，不過在找小巴的過程中意外發現一個睡著的司機



擋風玻璃前放了一份水果日報！！水果日報耶！在水果日報的發源地看見水果日報真有一種親切感 XDDD







































小巴的另一邊停滿了的士，深紅色跟地下停車場的灰暗顏色很搭，如果換成我們的小黃可能就太閃了 XDDD



前往中環地鐵站的途中，意外捕捉到這個提著銀色公事箱的型男



看他穿著皮夾克，深色牛仔褲，皮鞋，提著公事箱，還真的是挺帥的 @@



香港路上好多這種裝扮，看來我有空也該去搞個公事箱提提 XDDD



















本團的Girl4女攝影師米米，GF1好物呀 XDDD





























到了上環車站，我們的目標是蘭桂坊酒店



出站途中我們看到這個"召援專線"，召援 XDDD



以後如果有違反兒少條例第29條的當事人，叫他們辯稱是在尋求協助好了 XDDDDD



















這是我們住的房間窗外的景色 ^^ welcome to Hong Kong!!

















































香港是一個極度地狹人稠的地方，除了一路往上蓋的大樓外，狹窄的街道與爭相凸出的扛棒，就是道地特色



複雜到有時你得走到店家門口，才看的到他的招牌



放完行李後，我們沿著威靈頓街，一路往跟吳小璇約定的地點 — 翠華餐廳



在沒辦法聯絡彼此的外國，我們靠的就是事先約定跟準時到達地點



接下來就是明明事先約好，還要裝作"阿真巧，怎麼在這裡遇到你們～"



不過這次失敗了，因為吳小璇重感冒遲到又忘記到店裡面找我們，所以錯過了 XDDD



























































其實我想吃的是撒尿牛丸，雜碎麵跟超級無敵海景佛跳牆 XDDD



香港行拖的太久才寫，久到我已經忘記這些菜的名稱了，可是沒有忘記的是他們有多美味



至少我有一顆長得像糖心荷包蛋的，我想他一定適用火雲掌煎的吧 XDDD



至於在香港幾乎每餐必喝的奶茶，老實說不是很對我的口味，但不難喝也很有特色



我很好奇香港人喝他們自己的奶茶到底加不加糖







































在翠華酒足飯飽以後，我們按照行程，前往公車總站，搭車前往香港島的南邊 — 赤柱



穿梭在大樓間，偶而露臉的陽光反射在大樓玻璃上，似乎宣告著下午天氣會更好



我們正式開始港澳之旅 ^^

Dec 12 2009

+ 武陵。風景 +





一些隨拍的風景稍晚上菜囉 ^^













開了好久，也睡了好久，我們終於要進入武陵囉 ^^













人家說行車時頭手不要伸出車外，我是都伸出去了











進入武陵沒多遠，我們就被風景吸引，準備下車走走，我一開車門就看見今天第一抹紅 ^^



















































































































其實我自己並不是很清楚，第一個點地名叫做什麼 @@



大概是因為在國內的關係吧，國內的旅遊我大概都不會太在意地點名



反正比較容易到達 ^^ 國外的話我就會很注意了，因為事前也會做很多的規劃



到達這個不知名的地點已經是中午，頂光卻還是一片藍，這就是海拔高加上空氣乾淨的關係囉



由於最近拍人像可能比較希望有光影或者顏色的對比，所以不知不覺連拍風景也被影響了



(當然除了shine兩口子那張例外，那張叫做搞笑側拍照，搞個枯樹當背景，是要後製成哈力波特嘛 XDDD )



































































接下來我們到了渡假村附近，先停了車，走路到溪邊櫻花勾吻鮭的保育地



光線已漸西斜，穿過樹林縫隙間的陽光，讓舉目所見更有層次，就連溪水也清澈見底



一伙人在這裡隨意的抓拍人像，風景，最後還拍起鳥來



我還是繼續我的光影與顏色的遊戲 ^^























































等到太陽終於被山擋住後，我們一行人回到渡假村 check in



武陵富野渡假村真的是怕遊客無聊，住宿安排DIY竹筒飯的行程



還在走廊間設置了一堆益智遊戲



米米運氣很好的終於拼成一個正方體，而我只是隨便亂拼成一個好像尼斯湖水怪的東西



晚餐後原本想出來看看銀河，卻因為月亮太亮了，導致星星反而黯淡…..





















































第二天天氣依舊晴朗無雲，離開渡假村時，溫度計還顯示24度哩



我們緩慢的出發前往福壽山農場，可惜福壽山農場的楓葉狀況沒有武陵的好，遊客也很多



我們在農場旁的小吃攤隨便解決了午餐，路邊的野貓一直對我們凹凹叫 @@























































































離開福壽山農場後，我們繼續往高處走，前往算是本次旅行最後一站，天池



天池的光線，也是這趟旅行我覺得最美的



錯落在樹梢間，完完全全的光影，既柔和溫暖，又有足夠的強度



這種光線，怎麼拍都美，不拍也很美囉 XDDD



離開這裡後，我們就慢慢的下山，結束這趟難得的旅行



臨走前築楓的最後一拍，正好陽光也灑了下來



此時我除了覺得山在歡送我們以外，更覺得



【怎麼這時會是築楓站在那裡……XDDDDD】











感謝築楓安排行程，兩位車長辛苦的駕駛



其實我們後來還有再往更高一點去，景色更純粹，只是光線已經不夠美了



這也是明年想要開三天兩夜，武陵一路殺到埔里團的緣故



期待明年11月底12月初囉 ^^y



感謝賞圖 <(.)>






Dec 8 2009

+ 我們在武陵5℃ +





話說攝影師的宿命就是沒有自己的照片………….







自己的部落格也是，PO了好多篇，自己的照片好像沒有 = =|||



所以難得放上合照啦，1204-5由築楓帶團上武陵



還去了福壽山農場，梨山，據說差一點要開到清境了



這次有陽光男孩築楓在，一掃上次上清境的爛天氣



天空一整個爆藍呀~~



團員阿翔跟女朋友還拍起情侶照來，真是閃瞎我也 @@!!!



武陵富野渡假村真的是一個前不著村後不著店的地方



我們週五下午到渡假村附近，把附近的景點拍完後



太陽也差不多下山了，氣溫也開始急速下降(白天可是有24度呢!晚上就只有上面那張那樣….)



在渡假村用完晚餐後(坐圓桌吃把費，超怪的)



真的是沒事可做，所以跟米米一起去進行30分鐘的按摩囉



抒壓的感覺真好 ^^b



第二天早上我們就出發前往福壽山農場



真的是芭樂點人很多，楓葉的狀況也沒有很好



不過只要空氣好，對我來說哪裡都是天堂的啦 ~~~~



坐在路邊邊曬太陽，邊來一碗小吃攤切仔麵，真是一種奢侈的幸福 ^^



下午我們前往天池，約海拔2500公尺左右



彷彿越高，一切就越美



當一行人在天池旁的樹林裡拍米米時



我邊排隊等咖啡，邊看著陽光穿過樹梢，灑落在爬滿青苔的石頭上



遊客來來往往，每個人的髮絲光都是那麼的柔和又耀眼



我們再往高一點的地方去，一台藍色的野狼125，靜靜停在一片高山草原的旁邊



這絕美的暮色，深深印在我的腦海裡



(所以我要去買GP-1啦，這麼美的地方怎麼能不在我們的旅行地圖上紀錄下來 =皿= )



我們乘著翻過山頭的雲霧，不情願的回到都市裡……..









明年揪一團武陵到清境的好了，三天兩夜一路開過去 = =+




Nov 27 2009

+ 清境。人像。風景 +





清境，有羊 ^^









自從去年退伍後，上來清境住了幾天，便下定決心每年都要上來一次





乾淨而且冰涼的空氣，一直是我的最愛





這裡的民宿大多充滿歐式風格，更讓整個清境像童話世界一樣





如果有天真的海平面上升，我們現在住的平地都被淹沒的話





我想我會馬上逃到清境吧





去年上來時，都是超美的藍天，今年卻見到了完全不一樣的清境





飄小雨，不知道是山嵐還是霧，一直盤据在山頭，日夜不散





又是另外一種感覺，期待明年的清境會給我什麼驚喜囉 ^^



































































今年上清境前，順道繞到奧萬大去，那時天氣還很好



不過時機點太早了，所以當然沒有楓葉囉…….











































離開奧萬大，到達清境已經快下午兩點，吃飯前還有藍天



沒想到吃完飯後，整個清境已經被霧籠罩



不過遊客觀賞剪羊毛秀的興致還是不減 XDDD



每次陽光露臉的時間大概只有幾秒鐘，所以也就沒有拍什麼了………





















































這次住的地點是，香格里拉空中花園



去年發現他有調酒吧，但去年因為正在整修，所以沒喝到



想說今年一定要喝到，所以上來前就二話不說定了這家



結果！！！今年調酒師因為去旅展幫忙，不在山上……Orz



























整日的霧，讓我們想看銀河也沒機會了



只能記錄下這個灰濛濛的清境夜晚，外面的溫度大概是12度



民宿裡住滿了人，來來往往，好像一個家庭



晚上跟米米一起窩在房間看HBO，這就是我要的悠閒吧 ^^
