+ 行走於歐亞文化的樞紐 -- 2012 土耳其旅行企劃 +





2012年的年度大旅行計畫，我們決定去土耳其。

















6月9日從桃園機場出發，途經新加坡，並預計在伊斯坦堡Istanbul降落



一、6月10日至14日 — 伊斯坦堡 Istanbul



住宿：Hotel Turkuaz







伊斯坦堡交通圖







在14日晚上，我們將搭乘夜車，這也是我們旅行以來第一次在巴士上過夜，前往番紅花城Safranbolu。













二、6月15日至17日 — 番紅花城Safranbolu及其周邊Bartin、Amarsra，並且抵達黑海Black Sea



住宿：Kahveciler Konagi







番紅花城地圖







15日早晨抵達番紅花城後，將在附近短距離移動，並前往此行最北的黑海邊緣。







18日早晨我們將從番紅花城離開，先搭乘巴士往安卡拉Ankara，土耳其的首都，再立即轉乘飛機往Kayseri機場。













三、6月18日至21日 — 卡帕多奇亞地區Capadocia



18日下午抵達Kayseri機場後，隨即前往卡帕多奇亞地區兩大重要城市之一的格雷梅Goreme。



並且在卡帕多奇亞地區遊覽三日，21日下午再由Kayseri機場飛往Izmir機場。













四、6月21日至23日 — 伊茲米爾Izmir、賽爾柱克Selcuk



住宿：Urkmez Hotel







鄰近愛琴海，這裡是土耳其與希臘接壤的地區，將有非常多古希臘羅馬式遺跡可以探訪。



21日傍晚抵達賽爾柱克後，我們將在這裡停留三晚，並於24日上午從Izmir機場搭乘飛機返回Istanbul。













五、6月24日 — 返台



24日上午抵達Istanbul機場後，即直接轉乘飛機回台灣囉。

















細部行程將再繼續補充。

+ 旅行中的人像攝影分享 +





分享一些旅行中的人像攝影技巧，希望大家都能把一起旅行的家人、朋友們都拍得美美的 ：）








一、出發前，先做功課

我相信大多數人要出發以前，應該都會做旅遊目的地的功課，

不外乎就是上網找找別人的遊記、資料、書等等。

然而，如果企圖在旅行中拍攝比較特殊的人像，而不是一般的到此一遊照，

在做功課時就必須了解當地的氣候、風格、顏色、形狀等等，

有了這些資料，才能決定拍攝的時間、地點、服裝搭配，然後拍出合適、不突兀的照片。

做這類功課的方式，其實就是上Google、Flickr等網站搜尋圖片，

除了分析圖片外，也順便了解如何前往。

二、如果可能，盡量自助旅行

並不是說跟團拍不出好照片，但自助旅行的優點就是，

所有的行程、時間都在自己的掌控中。

拍攝旅行中人像攝影，自由度是非常重要的，

因為我們時常在某個定點看到美麗的風景、光線、角度，就會停下來拍攝，

有時甚至會花上15-20分鐘，如果是跟著團體走，恐怕就無法這樣隨意停留，

特別是你在遊覽車上時更是不可能。

另外，為了拍攝人像攝影，有些地方我們會不只去一次，

這也是大多數團體行動無法做到的。

自由度是一個重點，而拍攝的時間更是重要，

並非每天24小時都適合拍照，有些特別適合拍照的時間（清晨、黃昏），

旅行團不是在吃飯，就是在拉車，不然就是為了吃飯拉很遠的車，

這對於一個想把旅行中人像攝影拍得比較不一樣的旅行者來說，都是無法接受的。

另外，旅行團會帶你去的地方，都是所謂熱門景點，

但是越熱門其實越難拍，因為你只會拍到很多人頭而已，

而自助旅行，你可以選擇去人少，但是有特色的地方，這些地方都非常適合拍照。

簡言之，如果可以，盡量自助旅行。

三、看的出來是在哪裡拍的

寫這篇文章的目的，並不是要教大家如何外拍、如何拍Model、或讓大家成為職業攝影師。

目的在於提升到此一遊照的品質，讓他變得比較不一樣，變得更好一點，而且不難。

到此一遊照，或者可以說「看的出來是在哪裡拍的，有紀念性的」的照片，

重點當然在於要看的出來是在哪裡，如果看不出來是在巴黎拍的，還是在八里拍的，那其實在八里拍就好。

巴黎，羅浮宮，使用長鏡頭讓背景模糊



羅馬，聖彼得廣場，使用廣角鏡頭，帶入半個廣場

法國，聖米歇爾山，使用廣角鏡頭，帶入背景

巴黎，迦尼葉歌劇院，使用長鏡頭，帶入整片金碧輝煌

要表達「我有去過這個地方」，最快的方式就是跟知名地標合照囉。

至於取景可以有很多選擇，你也不見得一定要很靠近他。

另外，你也可以與當地特色拍照，不同的外國景色，

很容易就讓人分辨這張照片是在國內拍的還是國外拍的

南法，普羅旺斯，奔牛村，家家戶戶都有的爬藤植物

巴黎，左岸，靠近塞納河滿滿的攤販

威尼斯，前往Burano島的渡船頭，白黃兩色的設計令人印象深刻

羅馬，在義大利自助旅行，Tabacchi永遠是你的好朋友

四、由服裝、飾品，凸顯人物在畫面中的位置，並且適合當地風情

首先，有些地方有特殊的服裝限制，不能露肩、露腿，

不能穿涼鞋等等，請務必遵守，這是對文化的尊重喔。

然後，透過出發前做的功課，你可以將你的服裝、飾品，

搭配成符合當地的感覺，這樣可以很容易地將人物融合於當地。

希臘，米克諾斯島，海島風的穿著，顏色也正好配合窗戶

威尼斯，布拉諾島，充滿色彩的島嶼，黑色洋裝最能隨意搭配

威尼斯，充滿圖案的衣服，可以凸顯人物

另外，善於製造對比，透過顏色、構圖、甚至是最簡單的散景，

凸顯人物的所在，也是非常重要的。

羅馬，梵諦岡博物館，黃色的背袋在整幅畫面中，有特別凸顯的感覺

威尼斯，長鏡頭構成散景，是凸顯人物最簡單的方法

波蘭，格但斯克，長鏡頭壓縮背景，雖然人物較暗，反而凸顯人物

五、挑與眾不同的時間拍照&隨時都在拍照

一般來說，清晨大家都在睡覺，旅行團可能正在吃早餐然後去趕車，

晚上大家也都把相機收起來，但其實如果有足夠的體力、合適的器材，

挑選與眾不同的時間拍照，最能拍出特別的照片。

威尼斯，聖馬可廣場，想要拍攝無人的聖馬可廣場，大概也只有清晨了

南法，亞維儂，仍然保存良好的舊城牆，是亞維儂的一大特色

至於隨時都在拍照，畢竟一般人出國不容易，

雖然前面說拍照有最好的時間，但事實上隨時都在拍囉。

六、與當地人拍照

與當地人合照就不要太拘束了，只要不是吃豆腐，就大方地接受吧！

羅馬，熱情的老闆

羅馬，羅馬競技場，巧遇來自外地的女攝影師

希臘，米克諾斯島，全白的手工衣服店

七、選用的器材與建議

基本上，現在的數位相機，只要越貴越重，拍出來通常品質越好。

不過也同時造成體力的負荷。所以筆者建議，挑合適的相機，

當然數位單眼相機是最好，但也不一定要攜帶非常多的鏡頭，有時一顆就夠用了。

東京，上野公園，利用光影在地面上製造的圖形作為背景

東京，淺草寺，挑戰逆光，讓樹葉透光發亮

東京，淺草，簡單的填滿畫面，數大便是美

東京，風雷神門，以斜角構圖的方式，凸顯人物與燈籠的比例

東京，街頭，入夜後拍攝夜景，通常以廣角鏡頭比較方便，避免手震帶來影像模糊

上面五張東京的照片，都是用同一顆鏡頭拍的。

其實任何相機都可以拍，這裡只是說明數位單眼算是兼顧方便、品質，最好的選擇囉。

+ Hong Kong 2012 +





2012年的元旦假期，我們再次回到香港 — 與一大群友人 ^^

























隔了一年多的香港，這已經是我們第三次踏上這個地方



這次多了近十位好朋友，大家相約在香港碰面，一起跨年，吃好料，血拼



不過到最後，我自己的重點只在吃好料，香港煙火很一般般，也沒看到什麼好買的



於是，就把換的港幣都吃完了 XDDDDDDDDDD























晚上八點多的飛機，加上delay，我們到了旅館已經是半夜一點多



但原計畫毫無任何改變，所謂的原計畫就是 — 行李丟著馬上去吃翠華 XDDD



原本只要要問從酒店到中環翠華要做多久taxi，沒想到lobby服務生居然說，灣仔正好開了一間翠華呀



結果不到十分鐘的路程，奶油豬與香滑奶茶就在眼前了，搭配蔬菜麵線，原本感冒的症狀完全消除 XD

















心滿意足的回到飯店，已經是凌晨兩點多，lobby的聖誕裝飾還在



到隔天早上我們才可以大致欣賞房間的窗景，這次為了要看煙火，特別選了號稱可以看港景的房間



結果根本就沒有什麼view嘛，一晚六千多塊真是白花錢 T__T

















與其寄望沒有view的房間，還是把時間花在吃東西上面，比較划算



這整篇幾乎就是香港三日食記，一早在灣仔站附近的餐廳，吃的是奶茶、多士與雙煎蛋



吃完先去解決網路問題，香港這幾年來的無線網路似乎發達了許多



就算不買預付網卡，在任何貼有pccw的地方也能夠享受短暫的無線上網，不過還是受限於地點



而需要方便互相聯絡的我們，選擇去買一種98港幣的預付卡，每天最多扣28港，可以無線上網



只是有點不好找，雖然網站介紹7-11有得賣，實際上卻沒有，反而是在通訊行比較容易找到 @@









































其實此時離吃完早餐還沒過多久，但我們還是按照與熊貓夫妻的約定，到了東方小祗園



東方小祗園是去年到香港時發現的極好吃素菜，當時一吃驚為天人阿



不過這次吃的是分店，顯然跟上次吃的總店有點差距，還是說是我根本不餓的關係



叫一份四人桌菜，根本就吃不完，撐的要死 @@…..











這次到港並沒有特別安排行程，只有一個地方要去 — 大澳漁村



其他的時間就是吃吃喝喝加上血拼了，所以小祗園一結束，便走路到銅鑼灣時代廣場 — 每次到港必經之地



Zara已經連續幾次讓我們失望了，這次也沒例外，倒是Uniqlo比台灣便宜一點



看起來是該給台灣的Zara一次機會，說不定台灣挑的貨會比較好 Orz……





























由於今天是2011的最後一天，一行人在台灣就約好，6點所有人至油麻地C出口集合，一起吃飯過新年



只是沒想到這吃飯的地方還真的很有年味 XDDD



綺雯帶我們去吃有名的興記煲仔飯，他是用一個石鍋直接烤，搭配肉類



也有羅漢齋煲仔飯，所以對吃素的人來說也是ok囉



老實說米不是我喜歡的那種，但這是真正的香港小吃，非常道地^^

















煲仔飯食完後，綺雯帶我們在廟街逛大街，一直到這美部餐室附近，大夥兒停下來吃芝麻、花生、核桃糊



美部餐室的綠色窗框也是廟街有名的風景，底下則是夜市攤位，跟台灣差不多感覺^^

















沒想到吃著吃著，米米與綺雯就拿出生日蛋糕來了 @@



原來是米米前一天就偷偷請綺雯幫忙買蛋糕，但是蛋糕有夠難切



而且搭配芝麻糊真是一整個吃不下阿~



好笑的是生日快樂歌唱完中英文版後，還有廣東話版，芝麻糊店老闆還幫忙唱 XDDDD























接著就是廟街之旅，因為完全沒有帶大相機出來，所以都是用iPhone4s拍的



畫質…..當然就不多說了，可以拍就好 XD



整條廟街夜市非常熱鬧，大家還在經過涼茶店時特別喝了一杯



香港的涼茶也很有名，有區分很多種不同功用，同行友人有感冒症狀的，喝了一杯以後也稍微好了一點



其實應該買一些沖泡的回來泡的，但到最後卻忘記了 T_T





























大概走到佐敦站附近，大家已經決定要各自回旅館，不想在外面人擠人



今天晚上是跨年夜，照往例香港島的大樓會施放煙火，最佳的觀賞點當然是九龍或太平山



但人實在太多，也慢慢開始有人潮往那兩個地方去，加上如果去擠，可能看完要花一兩個小時才能回到旅館



所以我們老人團理所當然地放棄了 XDDD





























回到港島以後，距離跨年還有一些些時間，所以我們決定去做叮叮車，看看最遠可以坐到哪裡 ^^



乘坐的方向是從金鐘往上環，因為先前住的蘭桂坊在上環，所以那邊還算熟



沿路反而沒有什麼遊客，可能是集中到別地方去了吧，整個港島反而有些冷清



就連經過蘭桂坊時，路上也沒什麼人，只有冷風呼呼吹



坐叮叮車對我們來說，是港島最迷人的交通遊玩方式，因為便宜、又古老，大約有一百年左右的歷史



雙層並且有窗戶，可以瀏覽港島的風景，不僅是白天或夜晚都一樣



如果可以，通常我們是不坐地鐵的，那很無聊呀 ^^



































坐著坐著，直到我們覺得搞不清楚這是哪裡，應該要準備下車了



沒想到，不曉得是運氣還是職業病的緣故，一下車就是號稱香港最古老的婚宴會場 — 西港城



建於1906年的西港城western market，採用英國愛德華式建築，起先是海事處的辦公室



現在一樓則是餐廳，二樓以上是會堂，可以用作婚宴會場或國標舞表演場



這天還真的有人在婚宴，所以我們像勘景一樣四處入侵 XDDD

















第二天一大早，我們準備出發前往大澳漁村，一個要經過東涌站再轉巴士的偏遠地方



這次的香港行蠻幸運的，雖然一早天氣總是陰陰的，但九點多以後太陽就會露臉，接著便是好天氣



昨晚停在維多利亞港欣賞煙火的船，今早已經都全部撤退了



而我們就在灣仔站附近的茶餐廳，享受多士配奶茶配煎蛋的港式早餐 ^^





























一路睡到東涌站，綺雯已經在這裡等我們，額外的收穫居然是 — 東涌站外面就是一個超大的outlet mall @@



對於女生來說，這下子回來不愁沒地方去了，outlet的四周是高樓大廈，應該都是住戶



這麼密集的居住方式看起來真讓人心驚，約莫步行五分鐘，就可以看到前往大澳的11號巴士囉



沒想到想去大澳的人很多呢，整台車坐的滿滿的 @@

















約莫40分鐘的飆山路，而我們也在車上繼續補眠，大澳不知不覺地就到了

完成羅浮宮參訪後，我們要先去喝杯咖啡，再回頭拍羅浮宮夜景 ^^

這是一間1880年就開張的咖啡店，算算也有一百多年的歷史了



不過官方網站上面好像沒有歷史介紹，所以只能知道她歷史悠久







































我們到的時候好像已經快要打烊了，但店裡面還是有非常多客人



二樓已經休息，一樓仍然是坐滿的狀態，幸好有剩下一張四人座



坐定以後，女服務生為我們送上menu，深咖啡色的menu跟咖啡豆差不多



而環顧四周，其實這家店不只賣咖啡，還有賣茶，茶葉一櫃一櫃的



擠在小小的店裡感覺很溫暖，女服務生也說我們可以照相 ^^









由於不知道究竟哪一種好喝，所以隨便點點，在咖啡還沒來的時候



順便跟老闆買了幾包咖啡豆，後來帶回台灣，沒到半年就全部喝完了



簡約的咖啡杯造型配上一個葉子狀的盤子，還挺美的



當然也不能少了一杯咖啡後的巧克力，巧克力包裝紙上的圖案非常有趣



咖啡也一如預期的非常順口，在歐洲旅行，我最喜歡的就是拜訪有歷史的咖啡店囉



那裏總會給人一種溫馨，而且你很容易就可以想像一百年前的人



跟你一樣坐在這裡，外面有可能是馬車經過呢 ^^







































喝完咖啡以後，我們還想辦法解決了晚餐的問題，畢竟在巴黎要等到入夜，大概是在晚上九點多開始



所以上面三張照片的拍攝時間大概是晚上九點半以後了，今晚的光線出乎意料的美@@

而且整個光線以非常誇張的角度斜射進來，所有的人、建築物都染上柔和的暮光



這時捨捨與阿牛已經開始衝進羅浮宮巨大的廣場，而我跟米米還停留在街上



因為光線美麗的讓人震驚，而說不出話來……..

這時遠方橘紅色的雲，搭配穿越玻璃金字塔的光線，以及在玻璃上留下閃爍的反光



天ㄚ……在這倒數第二天晚上，巴黎終於彌補了我們前幾天的遺憾，給我們最精采的大景



特別是第一張照片，整座德農翼樓染上金、紅、橘色的光，我這輩子第一次看到這種光線



應該說，我這輩子第一次看到這種光線，打在這種建築上，我記得當時我在德農翼樓前站了好久……





























直射翼樓的陽光慢慢消失後，我們也退到靠近蘇利翼樓的地方，捨捨與阿牛架起腳架開始夜拍天空的色溫



此時整座羅浮宮也亮了起來，我因為沒有帶腳架，所以用手持的方式拍了幾張夜景跟人像



這時候的金字塔周圍還是有很多遊客、情侶，甚至稍晚還有拍攝婚紗的新人



米米也跟我最愛的德農翼樓合照，不過當時其實超冷的 XDDD









來稍微介紹一下羅浮宮的玻璃金字塔 Louvre Pyramid



羅浮宮的玻璃金字塔由華裔設計師貝隼銘I.M.Pei所設計



1984年，當時的法國總統密特朗決心要幫羅浮宮設計一個新的入口，因為舊的入口已經無法負荷參觀的人潮



貝隼銘有幸為羅浮宮此項改建擔任設計師，並且因為密特朗是個極度崇拜埃及的人



所以這座玻璃金字塔就這麼誕生了，位於金字塔下方的廣場則成為羅浮宮現在的主要出入口



1989年金字塔完成後，距今也已經超過了二十個年頭……









這座玻璃金字塔非常的有名，相傳剛建造完成時受到巴黎人的鄙視，認為為何要選金字塔這個主題



但後來人們也慢慢接受，現在已經是巴黎的指標建築之一



倒是蠻少人知道金字塔當時在密特朗的指示下，以666片玻璃構成，而666則是代表撒旦的數字 XDDD



另外玻璃金字塔也屢次出現在小說裏面，在丹布朗的小說達文西密碼中



蘭登在一開始就是經過金字塔進入羅浮宮，而最後的結尾也顯示所謂聖杯，就在金字塔底下的密室(當然沒這密室)



而在波西傑克森的作者另一著作—紅色金字塔中，也描述主角經由玻璃金字塔的魔法通道逃離怪物的追殺 ^^



























































就這樣，我們在金字塔待了好久，一直到已經是晚上十一點多接近十二點



天空的色溫終於消失後，我們才拖著疲累的步伐去坐地鐵返回公寓



然而，今晚的天空相信已經完全烙印在我們的腦海裡，羅浮宮給了我們最美麗的一天 ^^

















明天是在巴黎的最後一天，我們將在左岸閒逛，體驗巴黎最後的熱情與自由 ^^
