+ Eneloop colorful openbox +





由於目前有四顆燈加上一個電池包的關係，N年前買的Sanyo Eneloop電池有不夠用的跡象



所以~最近終於狠下心計畫要敗新電池啦~~~



原本計畫買一般的白色版本eneloop，然後拿奇異筆在上面標明今年購買



以免跟舊電池混用，傷到閃燈



上賣場逛壓逛，居然看到sanyo出了這組特別版 @@



不過要價也不低，一般白色版一顆大概是七八十塊，彩色特別版八顆要賣990 @@|||



這讓我我又猶豫了一兩個禮拜……..不過在前幾天突然又手殘就敗家了 T_T



於是繽紛到爆炸的彩色版Eneloop就出現在家裡 XDDD















為了要讓比較常用的SB900跟SB800還有電池包吃新的電池，一次狠心買了兩組……..Orz

















打開包裝，哇賽感覺很像小時候上美勞課用的蠟筆耶~~我的閃燈從此人生是彩色的啦~Q__Q



















列隊~N年前的白色eneloop就是班長了！！！！



















今天開始我們的任務就是支援火力！！！



















Panasonic PE28S，由於不是主要火力，所以使用舊的補給！！！



















第一主砲台，號稱可以zoom到200mm有如雷射光的SB900，火力滿檔~~~~~



















SB800出現了(剛剛一直在負責打光)，一樣火力滿檔！！！



















超級E罐SD8A也補給完畢！！(黑不啦機扁扁的電池包配上彩色電池真是超怪的XDDD)





















最後這張突然糟糕了 XDDD















感謝收看開箱文 : )

+ NIKON GP1 FIELD TEST & WINTER SNOW IN TAIPEI +

入手GP1以後，趁著米米週末下午在台北市晶華飯店外面廣場有表演活動





順便測試一下GP1在戶外的定位能力





GP1在室內的定位能力是不太好的，我在家中開機好久，都沒辦法抓到衛星





一直到我忘記我有開機這件事以後，他才終於定位成功 = =|||





不過在戶外倒是蠻快的，我想大概還不到20秒吧，GP1就已經定位成功囉 ^^





只是說在高樓林立的台北，GP1的準確度如何呢?





這直接關係到月底我們要去香港，到底可以期待多少的問題 @@





以下就提供8張照片，來看看GP1的能耐嚕





另外就是那天下午拍著拍著，台北市居然下起雪來了 @@!!!





有圖有真相~~~















這張定位到晶華旁邊民宅去了 = =|||















這張更扯，定位到民生西路馬偕醫院的後面去了 = =…..















這張拉回來了，不過拉過頭跑到老爺酒店前面馬路上 = =….|||















鐺鐺~~~終於回到了晶華飯店前面廣場 QQ















繼續維持在晶華飯店前面廣場 ^^y















又偏到老爺酒店去了…..Orz，是因為我在拍老爺酒店嗎?















持續定位老爺酒店中…..(翻桌)















最後一張又回到晶華飯店前面廣場了…..T_T



另外我敢發誓這張我絕對沒後製 =  =+











OK….GP1定位成功率是…..





最精確 3/8 = 37.5%，100張大概最多40張準…..





次精確不要偏太遠 7/8 = 87.5%，也就是偏到馬偕醫院那張除外…..





真不知道是該喜還是該悲了 @@





以後有機會再測試看看空曠地區





屋屋我的六位國父跟二十四位小朋友 T_T






Dec 15 2009

+ LOLO & GP-1 OPEN BOX +





敗家 XDD








自從北海道之旅回來後，就一直有想要把自己拍的照片的位置紀錄下來的想法





若真的要做這件事，首選大概就是Nikon自家的GPS紀錄器 —- GP1





可是網路上討論GP1的文章不多，而且經調查發現





GP1是吃相機本身的電 @@…





冷開機到定位有可能要花上一分鐘以上的時間 ~"~





價格實在是不曉得在貴什麼 Orz





因為他是接在原本接快門線的孔，所以還要再去買另一組快門線 = =|||





這些因素，讓我從六月龜毛到現在，在買跟不買之間猶豫著 Q___Q





直到12月初上武陵農場，武陵的美景成為壓垮駱駝的最後一根稻草





旅拍人像的特質就是，隨時有美景隨時停下來拍





往往路邊的一棵小樹，或者一台摩托車，搭配現場的光線就是絕美





而這些不起眼的小地方，又常常沒有什麼地標可以特定(就是路邊嘛 XDDD)





所以除非記憶力極強，否則過個幾年，照片的確實地點在哪裡早就忘了





這大概就是我想買GP1最大的原因了





不求很精確，不過大概知道照片是在哪邊拍的，透過Google地圖，可以有種回到那個地方的感覺 ^^









所以來開箱囉~











開箱囉~外號好奇貓的LOLO : 這是什麼這是什麼~@@













鐺鐺~內容物排排站，計有六樣



好奇LO : 這個電線看起來很新很好吃 (喜歡咬電線的貓)



尼 : LOLO你擋住我了啦 Orz













好奇LO : 不要開箱了啦~來陪我玩~ (把我的腳當貓抓板)













好奇LO : 不陪我玩，那我乾脆一屁股坐在你的GP1上面~



尼 : ………………………………………………………………………………….













把LOLO攆走之後，終於有一張正常的…….Orz













拍完開箱後，留下要使用的部分 ^^



好奇LO : 我又來了~打我壓笨蛋~



尼 : …………………………………………(握拳)













好奇LO : 我就是要把光擋~~住



尼 : …………………………………………….(集氣…界王拳….)













好奇LO : 不陪我玩我就搗蛋(壓)



尼 : 喂…..我都還沒開始用……













鐺鐺金剛合體~~線要接在快門孔，夾子要夾在相機背帶上面 ^^













與D3合體!!!看起來還ok，不至於造成不方便 yeah













鐺鐺~D3開機~GP1也開始閃紅燈搜尋衛星囉~~

















下一篇，GP1 Field Test~~~Rock ^^







LOLO下台一鞠躬~~~~

【敗家紀實】雨





2009年7月2日的晚上，練習拍雨






幾天的悶熱，在昨晚雨終於下下來，剛好興起了拍雨的念頭



目前的心得大概是



1.逆光或者側光拍



2.選擇較暗的背景



3.使用較小的光圈



去年6月的郭小靜









光這樣跟敗家記實好像無關 XDDDDDDD



其實是因為7/1入手小小黑 70-210mm f/4，補充長焦段



沒辦法，小黑五離我而去之後，某些活動要拍實在很痛苦 = =|||



本來在看105DC、135DC、180/2.8D、70-210/4



後來還是選個便宜的，等70-200/4G VR (望穿秋水呀…….)



快出吧泥坑！！
