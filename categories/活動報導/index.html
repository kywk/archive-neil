【活動報導】在一個夜黑風高的夜晚.....





在一個夜黑風高的夜晚….雲層漸厚….



失去月光的照耀….大地進入了一片黑暗….連樹木都顫抖著….感到不安…..



彷彿有什麼事情即將發生….有什麼不知名的物事即將出現…..







果然…當月亮緩緩失去光芒時….可怕的怪獸出現了….



同一時間….怪獸的背景音樂也響起…..



「有怪獸…有怪獸…有怪獸…纏著我…有怪獸…大怪獸…醜怪獸…黏著我….」







背景音樂響起的同時…怪獸張開了血盆大口….露出因為蛀牙而變成圓柱狀的牙齒….



旁邊的小鳥兒也被嚇得施展升旗式飛行法…想要逃離怪獸吐出來的口臭….



究竟誰能消滅怪獸，拯救這個世界呢？欲知詳情，請見下回分解~






上面是來亂的 XDDDDDDDDDD



昨天也就是2009.7.22，我見到了這輩子第一次日食 (這證明我還很年輕？)



為了迎接這個難得一見的天文奇景，我還特別請了一小時的假，到辦公室樓下的中正紀念堂觀賞



可惜昨天的天氣似乎不像氣象局預報的，受高壓籠罩萬里無雲



反而有很多雲層來來去去，有一大半時間太陽是被雲遮住的



不過也託雲層的福，透過雲層減光，我才能拍到一些像月亮的日偏食



這也算是不幸中的大幸吧



(磁片只能拿來肉眼觀賞方便，用來拍照真是超難用的 Orz )



早上有一則對岸的新聞報導，因為日食



造成山鳥齊鳴、猴子嘶叫、長頸鹿聚成一團不知所措、眾鳥提早歸巢



還有大陸民眾提著鳥籠一起去看日食，據說鳥兒變的比較不會叫



不曉得埃及有沒有像神鬼傳奇第一集一樣，大批蝗蟲鋪天蓋地而來 XDDDDDD



一次很特別的經驗，宇宙萬物真的有太多神奇的地方 : )

【活動報導】凡爾賽拜金女
